# GitHubFollowers

A feature-rich iOS app for searching GitHub users and exploring their followers. This project was implemented entirely in UIKit and serves as a showcase of my skills and understanding of UIKit development, navigation, and best practices. While based on a tutorial, all implementation is my own, demonstrating what I have learned.

---

## üöÄ Features

- Search for GitHub users by username
- View detailed user profiles
- Browse followers and following lists
- Navigate between users' follower networks
- Display user repositories and stats
- Custom alert dialogs throughout the app
- Smooth navigation using UINavigationController and UITabBarController
- Responsive layouts with Auto Layout
- Gesture recognizers for seamless UX
- Error handling with user feedback
- Custom reusable UI components
- Protocol-oriented delegate patterns
- Entirely UIKit-based (no SwiftUI, no Storyboards)
